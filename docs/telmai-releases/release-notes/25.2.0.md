25.2.0
======

Release date: 2025-04-14

## Product Changes

* New API for Data binning
* New connector: Iceberg
* New feature: Metrics Inspector (Only in Telmai Demo Environment)
* Bug fixes
  * Handling empty lists for Snowflake
  * Alerts not shown for AVG metrics
  * Custom Metrics: Page infinitely loads on adding same metric
  * Data API: `skip_investigator` flag explicitly passed does not override what is stored in connection
  * Issues connecting to Starburst sources
* Patch 1 bug fixes:
  * Issue creating rules with Trino sources

!!! important
    Post-update step must be run to update existing data binning configurations. Steps defined [here](25.2.0.md#post-install).
